# MyLittlePony-Data-Science-Project

This is a repository solely for my own personal project titled **“Who’s your favorite pony?”: An Analysis of My Little Pony Demographics Regarding Their Favorite Pony and Consumer Engagement Profiles**.

This data science project focuses on cleaning, processing, and analyzing consumer market survey data for the **My Little Pony: Friendship Is Magic (Generation 4)** franchise. Utilizing **R**, the study explores character favorites, customer demographics, and how much is spent by fans on merchandise.

## 📊 Dataset Profile
The project analyzes an simulated market survey dataset consisting of over 1,500 unique respondent records across 7 distinct attributes:
* `Respondent_ID`: The unique identifiers for each respondent.
* `Age`: The numerical age of the respondents (filtered to valid lifespans between 1 and 99).
* `Gender`: Categorical demographic marker for gender (Male, Female, Other).
* `Favorite_Pony`: Encompasses the standard Mane 6 as well as fan favorites outside the main group (e.g., Starlight Glimmer, Derpy Hooves, Princess Luna).
* `Fan_Segment`: Categorical profiling that identifies the consumer archetype. It is ordered (`General Public` → `Casual Fan` → `Child/Parent Collection` → `Brony`).
* `Annual_Spend_USD`: The continuous numerical metric representing financial investment in franchise merchandise.
* `Country`: The geographic location of the respondent.

## 🚀 How to Run the Project
1. Clone this repository into your local RStudio environment.
2. Ensure you have the needed packages installed.
3. Run the R script.
